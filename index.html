Build a structured prompt in seconds. Use form fields or dump-and-go paste boxes — whichever's faster. Paste the result into your PMTP Quote Builder Claude project.

Claude for Chrome is a free browser extension (with your existing paid Claude subscription) that reads any webpage you're on and gives you formatted output ready to paste into the dump fields below. It cuts hotel and tour research from minutes to seconds.

What it does for you

  • Open any hotel website, tour operator page, or supplier portal
  • Open the Claude sidebar in Chrome
  • Ask it to extract the details you need — formatted to drop into this tool
  • Copy → paste into the dump field. Done.

Install it (one time, ~3 minutes)

  1. Make sure you have a paid Claude subscription (Pro, Team, or Max). Log in at claude.ai if you haven't.
  2. Go to the Claude extension on the Chrome Web Store.
  3. Click Add to Chrome, then confirm.
  4. Pin the extension: click the puzzle-piece icon in your toolbar, then the thumbtack next to "Claude."
  5. Click the Claude icon in your toolbar and sign in with your Claude credentials.
  6. Grant the permissions it asks for. The sidebar opens.

Works with Google Chrome and Microsoft Edge. Not currently supported on Brave, Arc, or Safari.

Save the four shortcuts agents use most

In the Claude sidebar, click settings → Shortcuts → New shortcut. Save each of these so you can call them with /hotel, /tour, /flight, /transfer:

/hotel-extract
Read this hotel webpage and extract the following, formatted as plain text I can paste into a quote-building tool. Do NOT include any prices or rates from this page.

Output exactly this structure:

Hotel name: [name]
Location: [neighborhood, city]
Star rating / category: [if shown]
Room categories: [list each room category name only — no prices]
What's included: [breakfast, Wi-Fi, parking, etc.]
Highlights: [3-5 short bullets — pool, spa, restaurants, view, walking distance to X]
Address: [street address]
Website: [the URL of this page]

Keep it tight. Skip marketing fluff. Skip prices entirely.
/tour-extract
Read this tour operator page and extract the following, formatted as plain text I can paste into a quote-building tool. Do NOT include any prices or rates from this page.

Output exactly this structure:

Operator: [Trafalgar / Globus / etc.]
Tour name: [full tour name]
Duration: [e.g., 14 days / 13 nights]
Itinerary highlights: [list day-by-day or city-by-city — keep it concise]
What's included: [meals, transportation, accommodations level, guides, included experiences]
What's not included: [optional excursions, tips, flights, etc., if mentioned]
Pricing tiers: [list tier names only — Standard / Premium / Deluxe — no prices]
Source URL: [the URL of this page]

Skip prices entirely. Keep the itinerary section to one line per day or city.
/flight-extract
Read this flight search result or booking page and extract the following, formatted as plain text I can paste into a quote-building tool. Do NOT include any prices.

Output exactly this structure:

Airline: [carrier name]
Route summary: [e.g., IAH → ATH nonstop]
Outbound: [date, departure airport + time → arrival airport + time]
Return: [date, departure + time → arrival + time]
Cabin class: [economy / premium economy / business]
Baggage: [if shown]
Notes: [refundability, mileage-eligible, layover details, anything relevant]

Skip the price entirely.
/transfer-extract
Read this transfer / private driver / shuttle page and extract the following, formatted as plain text I can paste into a quote-building tool. Do NOT include prices.

Output exactly this structure:

Service: [private transfer / shared shuttle / private driver / etc.]
From → To: [pickup → dropoff locations]
Vehicle type: [if specified]
Capacity: [number of passengers]
Duration: [estimated travel time]
Inclusions: [meet & greet, child seats, water, etc.]
Provider: [company name]
Source URL: [the URL of this page]

Skip prices entirely.
/photos-extract
Find every image on this page that shows the actual property, restaurant, tour, or experience (not logos, icons, banner ads, or stock illustrations). For each image, give me only the direct image URL — one per line, no labels, no descriptions, no commentary.

Skip:
- Site logos and brand marks
- Social media icons
- Banner ads or promotional graphics
- Map screenshots
- User profile photos / avatars
- Any image hotlinked from these blocked sources: trvl-media.com, hotels.com, vrbo.com, muscache.com, encrypted-tbn0.gstatic.com (Google Images cache)

Output format — just the URLs, like:
https://example.com/photo1.jpg
https://example.com/photo2.jpg
https://example.com/photo3.jpg

If you find more than 15 usable images, give me the 15 best ones (variety: exterior, rooms, restaurants, amenities, views). Order them most-to-least useful.

How to use during a quote

  1. Open the supplier page in Chrome
  2. Open the Claude sidebar
  3. Type /hotel (or /tour, etc.) and hit enter
  4. Copy the output
  5. Come back to this tool, paste into the appropriate dump field
  6. Add the cost and supplier yourself in the cost/supplier fields — the extraction never includes prices, by design

⚠ Never let extracted content fill in your cost fields. Rates always come from your sourcing — wholesaler, TAAP, hotel direct — not from public webpages. The shortcuts are designed to skip prices entirely so you can't accidentally paste a public rate into the calculator.
